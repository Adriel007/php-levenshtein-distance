O agrupamento K-means é um dos algoritmos de aprendizado de máquina não supervisionados mais simples e populares.

Normalmente, algoritmos não supervisionados fazem inferências a partir de conjuntos de dados usando apenas vetores de entrada sem se referir a resultados conhecidos ou rotulados.

AndreyBu , que tem mais de 5 anos de experiência em aprendizado de máquina e atualmente ensina suas habilidades às pessoas, diz que “o objetivo do K-means é simples: agrupar pontos de dados semelhantes e descobrir padrões subjacentes. Para atingir esse objetivo, o K-means procura um número fixo ( k ) de clusters em um conjunto de dados.”

Um cluster refere-se a uma coleção de pontos de dados agregados devido a certas semelhanças.

Você definirá um número alvo k , que se refere ao número de centróides necessários no conjunto de dados. Um centróide é a localização imaginária ou real que representa o centro do cluster.

Cada ponto de dados é alocado para cada um dos clusters por meio da redução da soma dos quadrados dentro do cluster.

Em outras palavras, o algoritmo K-means identifica o número k de centróides e, em seguida, aloca todos os pontos de dados para o cluster mais próximo, mantendo os centróides o menor possível.

Os 'meios' no K-means referem-se à média dos dados; isto é, encontrar o centróide.
